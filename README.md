# The Celestial Archive · 天穹秘典

A local-first bilingual 78-card tarot reflection app for a small friend preview.

## Download

[Download the latest friend preview](https://github.com/ted0103/celestial-archive/releases/latest)

1. Download `celestial-archive-friend-preview-v0.1.0.zip`.
2. Unzip it and open `START-HERE.txt`.
3. On macOS, run `python3 -m http.server 47831` inside the folder.
4. Visit `http://localhost:47831`.

The core app works without hand tracking. The optional gesture pack is prepared from Settings.

## Privacy

Readings remain on the device unless the user explicitly saves or exports them. Camera hand tracking is optional. This is a reflective experience, not medical, legal, financial, or predictive advice.
