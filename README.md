# The Celestial Archive · 天穹秘典

A local-first bilingual 78-card tarot reflection experience for friends.

## Open or download

- **iPhone, iPad, Android, and desktop:** [Open The Celestial Archive](https://ted0103.github.io/celestial-archive/)
- **All installation options:** [Open the illustrated download guide](https://ted0103.github.io/celestial-archive/download.html)
- **Offline ZIP:** [Download the latest friend preview](https://github.com/ted0103/celestial-archive/releases/latest)

### iPhone and iPad

1. Open the [direct website](https://ted0103.github.io/celestial-archive/) in Safari.
2. Tap **Share**.
3. Choose **Add to Home Screen**, then **Add**.

The website is the iPhone/iPad version. Apple devices do not use Android APK files.

### Android

1. Open the [direct website](https://ted0103.github.io/celestial-archive/) in Chrome.
2. Open Chrome's menu and choose **Add to Home screen** or **Install app**.
3. An APK download will also appear under the [latest release](https://github.com/ted0103/celestial-archive/releases/latest) when available.

### Windows

1. Download `celestial-archive-friend-preview-v0.1.0.zip` from the [latest release](https://github.com/ted0103/celestial-archive/releases/latest).
2. Right-click the ZIP and choose **Extract All**.
3. Open PowerShell inside the extracted folder.
4. Run `py -m http.server 47831` (or `python -m http.server 47831`).
5. Open [http://localhost:47831](http://localhost:47831) in Chrome or Edge.

### macOS

1. Download and unzip `celestial-archive-friend-preview-v0.1.0.zip`.
2. Open Terminal inside the extracted folder.
3. Run `python3 -m http.server 47831`.
4. Open [http://localhost:47831](http://localhost:47831).

The core app works without hand tracking. The optional gesture pack is prepared from Settings. Camera access requires the HTTPS website or a local server; opening `index.html` directly with a `file://` URL is not supported.

## Privacy

Readings remain on the device unless the user explicitly saves or exports them. Camera hand tracking is optional. This is a reflective experience, not medical, legal, financial, or predictive advice.
